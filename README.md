# bulk_rename_files
This is a CLI tool for bulk renaming of files at once.

## Install

```sh
cargo install --git https://github.com/shinshin86/bulk_rename_files.git
```

## Usage

```sh
bulk_rename_files <dir_path> <output_file_name (No extension is required.)>
```
